Flat Universe

Make This Universe Alive!

How to Play?

LeftClick : Make a New Star from Cursor | Up/Down Arrow : Increase/Decrease Power of Star
C : Set Power 0 of Cursor | B : Reverse Power of Cursor | Enter : Make a New Material System

GUI Stands For | MaterialLife : Life of Each Material | Nparticle : Material Gen per System | Slife : Life of Each Material System | MaterialMass : Mass of Material of MaterialSystem
*Warning* Changes to the gui will not affect the current system.
